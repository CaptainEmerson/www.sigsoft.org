This dissertation presents a scheme for adding tools to a programming environment so as to achieve and maintain an advanced and uniform user interface throughout the environment. This includes the following.

  • A 'tool addition scheme' with the following properties: it    requires simple tool form, allows easy tool addition, allows easy tool composition, and achieves advanced, uniform user interface to a growing and evolving tool-set.
  • An 'Ada shell' with the following properties: it provides a convenient user interface to tools, provides a sufficient user interface to the simple tool form, is uniformly extensible and customizable, and can provide alternate command styles (e.g., 'command-language,' 'menu-selection') to same tool-set.
  • Three distinct command interface styles including 'command-language,' 'menu-selection,' and 'character-action.'  Each of these interfaces may be completely automated using only the information provided by the specifications for Ada packages.
  • An 'Ada program entering tool' with the following properties: it is complete for Ada, is uniform because it uses plain text and prefixes, greatly reduces the number of key-strokes necessary to enter Ada programs, and allows Ada to be a suitable command language.

Each of these is presented along with examples and scenarios for various interactions with the environment.  This scheme has been demonstrated by implementation of the programming environment, 'Arcturus,' and the Ada Shell, 'Adash.'
